金庫は大変満足のいく中身を持っていることが分かりました。 それは現在の紙幣ではなく製造に金銀を使用された真に価値あるものです。 それらの上に貴重な宝石や宝飾品などのほか多数の高価な品がありました。 あなたはそれら全てを持って帰りたいと強く願いましたが、 残念ながらあなたの雇った船員たちはすでに死んでいます。 あなた自身で運べるのは宝の一部だけです。 よろしい、ならばそれらのうち最も価値あるものを持っていきましょう!

入力として金庫の中身を以下の書式で受け取ります。 {'golden coin': {'price': 100, 'weight': 50, 'amount': 200}, 'silver coin': {'price': 10, 'weight': 20, 'amount': 1000} , 'ruby': {'price': 1000, 'weight': 200, 'amount': 2}} 価格 ( price ) はあなたの国での貨幣の標準的な単位で測った値です。 重量 ( weight )はグラム単位です。量 ( amount ) は数量です。
加えて重量制限がキログラムで与えられます。 これを越えると運べません。

あなたの仕事は総重量が重量制限を越えない範囲でこれらの中からできるだけ高い価値の宝物を集めることです。 答は文字列のリストとして返してください。 例: ['golden coin: 150', 'silver coin: 700', 'ruby: 2'] 常に 3種の宝物 ( 金貨 ( golden coin ), 銀貨 ( silver coin ), ルビー ( ruby ) ) があります。 同じ順番で返してください。 もしある種の宝物が選ばれなかった場合 ( ある宝物を 0個とするのも可能です ) 答に含めないでください。

入力: 宝物の情報 ( 価値, 重量, 数 ) の辞書と重量制限

出力: あなたが持っていく全宝物 ( 文字列のリスト )
